# **App Name**: Shiva Shakthi Gold Assistant

## Core Features:

- Admin Price Update: Secure admin interface to update live gold prices (24K, 22K, 18K) and configure markups, taxes, and other calculation rules.
- Real-time Price Display: Display the Live Gold Price and Admin Gold Price with the last updated timestamp on the user login/home page.
- Advanced Gold Calculator: A comprehensive calculator with inputs for gold weight, purity, making charges (per gram/fixed), wastage (percentage/grams), GST/tax, stone price, and discount; clear breakdown of costs.
- Instant Recalculation: Automatically recalculate output values in the gold calculator based on changes to the inputs.
- Print & Invoice System: Enable users to print calculations in a format including the Shiva Shakthi Jewellers logo, date, gold details, and a full price breakdown.
- Daily Price Change Log: AI tool to intelligently maintain a history of daily gold price changes for admin review and auditing purposes.
- Admin Dashboard Summary: Display a summary dashboard for the admin showing today's gold rates and the number of calculations performed.

## Style Guidelines:

- Primary color: Deep gold (#D4AF37) to reflect the luxury of gold jewelry.
- Background color: Off-white (#F8F8FF), a desaturated gold color scheme.
- Accent color: Dark Brown (#8B4513) for interactive elements, creating a classic jewelry feel.
- Headline font: 'Playfair' serif, lending an elegant and fashionable style for headings.
- Body font: 'PT Sans' sans-serif, combining modern clarity and warmth for information.
- Use sleek, minimalist icons to represent different features and options.
- Subtle animations to indicate recalculations and data updates.
